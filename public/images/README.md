# Imágenes para el sitio

- **Logo:** El logo está en `/logo.png` (raíz de `public`).
- **Instagram / menú:** Para que las fotos del menú y la galería se vean en el despliegue, coloca aquí las imágenes en `instagram/` con los mismos nombres que usa el código (ej. `pukadogcolombia_DUERoqCksP1.jpg`).

Las rutas en el código usan `/images/instagram/nombre.jpg`, que Next.js sirve desde `public/images/instagram/`.
